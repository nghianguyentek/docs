# UI design
## Common Principles
### Audience-Oriented
- Content
  - Clarity: easy to understand and make decision
- Presentation
  - Easy to use: reading, navigation, and searching
### Simplicity
- Content:
  - No redundant
  - Direct to the desired action
- Presentation
  - Colour:
    - The number of colours should be 3-5.
  - Font:
    - Easy to read
### Consistency
- Content
- Presentation
  - Layout:
    - Symmetry
  - Font
  - Colour palette
### Portability
- Responsive
