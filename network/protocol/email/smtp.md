# SMTP (Simple Mail Transfer Protocol)
<pre>
SMTP is a text-based protocol.
The server listens at port 25.
</pre>
## Commands
- `EHLO`
- `MAIL FROM`
- `RCPT TO`
- `DATA`
- `QUIT`
