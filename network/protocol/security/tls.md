# TLS (Transport Layer Security Protocol)
## Purpose
Provide security for communication between two endpoints.
## Requirement
Underlying transport is reliable and sequential.
## Characteristics
- Properties:
  - Authenticated
  - Confidentiality
  - Integrity
- Components:
  - A handshake protocol
  - A record protocol
## Version
- 1.2 (Obsoleted)
- 1.3 (Current)
