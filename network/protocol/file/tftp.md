# TFTP (Trivial File Transport Protocol)
<pre>
A <a href="udp.md">UDP</a>-based file transfer protocol
## Characteristics
- No authentication
- Package types: RRQ (Read ReQuest), WRQ (Write ReQuest), Data (up to 512 bytes), ACK, Error.
