# UDP (User Datagram Protocol)
<pre>
One of standard transport protocols of IP suite.
</pre>
## Characteristics
- Unreliable: no timeout, no acknowledgement, no retransmission.
- Unconnected
## Header
- Source port: 16 bit
- Destination port: 16 bit
- Length: 16 bit
- Checksum: 16 bit (Internet checksum)
